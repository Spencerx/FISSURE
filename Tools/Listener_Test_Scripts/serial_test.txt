
Create virtual ports (data sent to /dev/pts/3 will appear on /dev/pts/4, and vice versa):
	sudo socat -d -d pty,raw,echo=0 pty,raw,echo=0
	
To prevent password prompts at the HIPRFISR (temporary solution)
	sudo chmod 666 /dev/pts/5
	
Set the Listener Serial Port: /dev/pts/3 (or whatever is listed)

Send Test Data to the Paired Port: /dev/pts/4
	echo '{"sensor_node_id": 0, "alert_text": "Simulated alert!"}' > /dev/pts/4
	
		
	
	
Real serial device:
	echo '{"sensor_node_id": 0, "alert_text": "Serial port alert!"}' > /dev/ttyUSB0
